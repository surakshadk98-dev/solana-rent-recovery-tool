# Solana Rent Recovery Tool 🚀

## 📌 Problem Statement

On the Solana blockchain, token accounts require rent (SOL) to exist.
Even empty accounts can hold locked SOL which users often forget to recover.

---

## 💡 Solution

This tool scans a wallet and identifies empty SPL token accounts, allowing users to close them and recover locked SOL.

---

## 🚀 Features

* Connect Phantom wallet
* Scan empty token accounts
* Calculate recoverable SOL
* Close accounts in one transaction

---

## 🛠 Tech Stack

* JavaScript
* Solana Web3.js
* HTML/CSS

---

## ▶️ How to Use

1. Open the app
2. Connect wallet
3. Click "Scan Accounts"
4. Click "Recover SOL"
